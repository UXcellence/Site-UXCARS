# 🚗 UXCARS

Site institucional da **UXCARS**, desenvolvido como projeto acadêmico para apresentar a concessionária, seus veículos elétricos e serviços, com foco em design moderno e experiência do usuário.

---

## 📋 Sobre o Projeto

A UXCARS é uma concessionária fictícia especializada em carros elétricos. O site foi criado para oferecer uma vitrine digital da loja, com seções de apresentação, catálogo de veículos, quiz interativo e informações de contato. O slogan **"Dirija Seu Futuro"** reflete a proposta da marca: tecnologia, design e excelência em um só lugar.

---

## ✨ Funcionalidades

- **Slideshow automático** — Banner na Home com troca automática de imagens de 4 modelos a cada 3 segundos.
- **Catálogo de veículos** — Exibição de 6 carros com foto, nome, preço e quilometragem, com botão de orçamento.
- **Quiz interativo** — 10 perguntas para descobrir qual carro elétrico combina com o perfil do usuário, com resultado personalizado e imagem do modelo recomendado.
- **Seção "Sobre"** — Apresentação da história, tecnologia e diferenciais da UXCARS, com link direto para o catálogo.
- **Rodapé com contato** — E-mail e telefone em todas as páginas.

## 🚀 Como Executar

1. Clone ou baixe o repositório:
   ```bash
   git clone https://github.com/UXcellence/Site-UXCARS.git
   ```

2. Navegue até a pasta do projeto:
   ```bash
   cd Site-UXCARS
   ```

3. Abra o arquivo `index.html` no seu navegador.
